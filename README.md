In contrast to conventional neural networks, this project presents a self-pruning neural network that addresses practical limitations, including memory and computation requirements. The proposed network eliminates unnecessary connections from the network by employing the PrunableLinear layer while training instead of performing pruning post-training. For each weight in the network, there exists a gate parameter learned during training. This gate parameter is transformed through the sigmoid function, generating values ranging from zero to one. The gate parameters act multiplicatively with the weight to control their importance. Additionally, the training procedure includes L1 regularization on the gate parameter along with cross-entropy loss. In this regard, the network learns to prune unimportant weights during training, leading to an efficient and sparse architecture.
The network was trained and evaluated using the CIFAR-10 dataset for varying sparsity coefficient (λ) values to investigate the relationship between accuracy and sparsity. Classification accuracy, sparsity percentage, confusion matrix, and heatmap visualization of gate parameters and pruning masks were used to evaluate the model. An increase in the λ value leads to higher sparsity at the expense of accuracy. The proposed model provides more insight regarding classification performance and sparsity of each class using the confusion matrix and heatmap visualization.
